# My CMU Game Website Plan

- **Project Title:** My CMU Game Website

---

# 1. CMU Game Selection

## Game Title
My game is called **Tic Tac Toe**.

## Link to the Game
[Play My Game Here](https://academy.cs.cmu.edu/sharing/floralWhiteAnt256754)

## Short Description
This is the classic 2 player game where x and o take turns filling in squares in the grid.
---

# 2. Website Content Plan

## Main Content I Want on My Website
My website will include:

- A large title banner for the game
- A short description explaining how the game works
- Screenshots from gameplay
- A playable embedded version of the game
- Instructions for controls
- A short “About the Developer” section


## Additional Features I Might Add
- Animated buttons
- Hover effects
- Background music
---

# 3. Layout Design

## Basic Layout Sketch

------------------------------------------------

GAME SCREENSHOT OR BANNER
------------------------------------------------

ABOUT THE GAME | HOW TO PLAY | HIGH SCORES
------------------------------------------------

PLAYABLE GAME AREA
------------------------------------------------

ABOUT THE DEVELOPER
------------------------------------------------

FOOTER
------------------------------------------------

## Layout Notes
- The navigation bar will stay at the top.
- The playable game will be centered on the page.
- Information sections will appear below the game.
- The website should work well on both desktop and mobile screens.

# 5. Tone and Style

## Overall Tone
The tone of my website will be:
- Fun
- Classy

## Writing Style
The writing on the page will:
- Use short paragraphs
- Include exciting action words
- Encourage players to keep things casual
- Feel welcoming and easy to understand
