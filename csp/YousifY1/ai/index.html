📖 About the Game

The High-Five is an interactive game where your decisions matter. Each choice you make affects the stick figure's mood. Can you keep them happy, or will they leave upset?

🎮 How to Play

Controls: Click or tap the buttons to make choices. Each decision changes the stick figure's emotion. Try different combinations to discover all outcomes!

About the Developer

Hello! I'm a student at Carnegie Mellon University studying game design and interactive media. I created The High-Five to explore how player choices can shape emotional storytelling in games.

This project taught me about game mechanics, player engagement, and how small interactions can create meaningful experiences.
