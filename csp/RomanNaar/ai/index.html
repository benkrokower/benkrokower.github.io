About This Project

My games are part of my fourth creative task and are recreations of Geometry Dash ship gameplay.

Inspiration

This project was inspired by Geometry Dash, originally made by RobTop. I wanted to recreate the ship flying mechanics with fun challenges.

About the Class

This project was completed as part of a creative task in my CMU class. The goal was to explore coding and game design concepts in a hands-on way.

Game Screenshots / Explanations

There are two variations of the game: a challenge mode where two walls converge from top to bottom, and a survival mode where blocks come from the left side.

Coding Challenges
