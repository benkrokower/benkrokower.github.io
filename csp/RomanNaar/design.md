# My CMU Game Website Plan

# 1. CMU Game Selection

## Game Title
My games are part of my fourth creative task and are recreations of **Geometry Dash ship gameplay**.

## Link to the Game
[Play My Game Here](https://academy.cs.cmu.edu/sharing/moccasinDog202923)

## Short Description
This project includes the recreation of the ship flying in the game Geometry Dash, originally made by RobTop. There are two variations of the game:  
1. A challenge where two walls converge from top to bottom, testing your straight flying abilities.  
2. A survival mode where you dodge blocks coming from the left side of the screen.
